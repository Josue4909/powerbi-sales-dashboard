# Power BI Sales Dashboard
This project demonstrates a basic Power BI dashboard designed to analyze sales trends and product performance.

## Tools Used
- Power BI
- Excel / CSV data
- Business intelligence visualizations

## Dashboard Features
- KPI cards for total sales and average sales
- Monthly sales trend chart
- Product category breakdown
- Filters for interactive analysis

## Purpose
This project demonstrates dashboard development and business analytics skills by transforming raw sales data into visual insights.
